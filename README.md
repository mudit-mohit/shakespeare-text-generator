## Overview
LSTM-based text generator trained on Shakespeare's complete works from Project Gutenberg.

## Features
- **Basic LSTM Model**: Simple architecture for quick training
- **Advanced LSTM Model**: Bidirectional layers for better quality
- **Temperature Control**: Adjust creativity of generated text
- **Project Gutenberg Dataset**: Complete Shakespeare works

## Models
- **Basic**: 2 LSTM layers (128 units each), Embedding layer
- **Advanced**: Bidirectional LSTM, 3 layers (256, 256, 128 units)

## Usage
1. Open `LSTM_Text_Generator.ipynb` in Google Colab
2. Run all cells sequentially
3. Choose between Basic and Advanced models when prompted

## Dataset
Uses the complete works of William Shakespeare from [Project Gutenberg](https://www.gutenberg.org/ebooks/100).
